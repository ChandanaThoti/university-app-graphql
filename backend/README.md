## University backend app


## Description
- This is a backend app university with students, courses and faculty where api managed by graphQL.

## Tech Stack

- **Express** with Typescript
- GraphQL (for api handling)
- Apollo Server (for graphQL server)
- Node.js (for execution)

## Installation

```bash
cd backend
```

```bash
npm install
```

## Start Project

1. **Start the application** using:

```bash
   npm run build      # Build the application
```

```bash
   npm start       # Start the application
```
